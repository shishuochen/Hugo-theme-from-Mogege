## 关于主题

从19年接触这个主题，迄今为止大大小小更改了很多地方，从而使其符合自己的口味


里面除了原来拥有的功能外，添加了很多可自定义内容,主要包括

- 评论系统更改
- 代码块样式
- 图片灯箱及caption优化
- ......

具体跟多细节内容可以查看：

- [Hugo 博客自定义优化 II](https://shishuochen.github.io/2022/cpvuqozuc/)
